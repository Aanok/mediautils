# Requirements
### h265:
* ffmpeg

### audio2opus:
* ffmpeg
* coreutils

### cuesplit:
* shntool
* cuetools
* flac
* ffmpeg
* monkeys-audio for ape/cue (from deb-multimedia repository)

These are all Debian packages. Please note the scripts strictly require Bash.
